# Crypto News Sentiment Playground

Interactive widget for crypto news sentiment inference using:
- News text
- BTC price now
- Fear & Greed (FnG) index
- FnG classification

Outputs:
- Sentiment (Bullish/Bearish)
- Score (-1 to +1)
- Confidence (0 to 1)
- Probability of up move
